# Tickets (Player-Reported Issues)

Track issues reported by players from Steam Workshop, Discord, or other channels.

## Template

Filename: `TICKET-[number]-[short-name].md`

```
# TICKET-[number]: [Title]
## Status: New | Investigating | Confirmed | Fixed | Cannot Reproduce | Won't Fix
## Source: Workshop | Discord | GitHub
## Reporter: [Player name / anonymous]
## Date Reported: [YYYY-MM-DD]
## Version: [target version]
## Description
[What the player reported]
## Steps to Reproduce
[If provided or discovered]
## Investigation Notes
[Findings during investigation]
## Resolution
[Once resolved]
```
