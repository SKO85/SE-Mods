# Features

## Template

Filename: `FEAT-[number]-[short-name].md`

```
# FEAT-[number]: [Title]
## Status: Proposed | Approved | In Progress | Done | Rejected
## Priority: High | Medium | Low
## Summary
[One-line description]
## Motivation
[Why is this needed?]
## Design
[How it works, settings, UI changes]
## Files Affected
[List of files that need changes]
## Testing
[How to verify]
```
