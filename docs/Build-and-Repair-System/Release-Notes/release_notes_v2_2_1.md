## Bug Fixes

- Fixed an issue that incorrectly blocked grinding of owned grids inside Admin Safe-Zones when the target blocks had no ownership. Safe-Zone checks now correctly compare the relationship between the Build and Repair block's owner and the target block.
- Fixed a mod initialisation error that appeared in the control panel info panel after reconnecting to a server.
