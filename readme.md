# SKO's Space Engineers Mods

A collection of mods for [Space Engineers](https://store.steampowered.com/app/244850/Space_Engineers/), maintained by SKO85.

---

## Mods

### Nanobot Build and Repair System

Automatically welds, repairs, grinds, and collects floating objects within range. Fully configurable per-block through the terminal and server-wide through a `ModSettings.xml` file.

## Contributing

1. Fork the repository or create a branch named `fix/<fix-name>` or `feature/<feature-name>` from `main`.
2. Implement and test your changes.
3. Submit a Pull Request. Keep PRs focused on a single fix or feature to make review straightforward.
4. PRs undergo automated review by Codex before a final review by SKO.

Once accepted, changes are merged into `main` and deployed to Steam.
